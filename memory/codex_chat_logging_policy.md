# Codex chat logging policy

Foydalanuvchi Windows 11 dagi Codex ishlarini GitHub'dagi `Muslimatun` reposiga yozib borishni
xohlaydi.

## Qoidalar

- Muhim ish, buyruq, fayl o'zgarishi yoki xulosa bo'lsa, `sessions/YYYY-MM-DD-*.md` fayliga yoz.
- Har bir logda kamida quyidagilar bo'lsin:
  - maqsad;
  - qilingan ishlar;
  - muhim buyruqlar;
  - natija yoki blocker;
  - keyingi qadam.
- Ish tugaganda o'zgarishlarni commit va push qilishga harakat qil.
- Agar push uchun login/token yoki network muammosi chiqsa, foydalanuvchiga aniq ayt.
- "Oxirgi qilingan ish nima edi?" degan savolda avval Git tarixini va `sessions/` loglarini tekshir.
